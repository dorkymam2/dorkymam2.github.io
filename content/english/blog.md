Welcome to the Physics Lair discussion board! 

If you are stuck on a problem set or need a concept explained, drop a comment below. Since we are using GitHub Discussions, you can format your questions using Markdown and write math equations using LaTeX.

### How to type math:
Just wrap your equations in `$$`. For example, writing the kinematic equation:
`$$ x = x_0 + v_0 t + \frac{1}{2} a t^2 $$`

Will render like this:
$$ x = x_0 + v_0 t + \frac{1}{2} a t^2 $$
